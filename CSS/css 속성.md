- 유동 속성
	- float : 위치 지정
		해제는 clear : both 또는 부모 컨테이너에 overflow : hidden
		- left : 컴포넌트를 왼쪽부터 배치
		- right : 컴포넌트를 오른쪽부터 배치
	- clear : float의 영향을 지우는 속성
		- both : float이 left든 right든 적용
