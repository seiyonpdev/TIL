- 대입 연산자 : **=**
- 사칙연산 연산자 : **+ - \* / %**
- 연결 연산자 : **+**
	- 두 개 이상의 문자열을 붙이는 연산자
- 복합 연산자 : **+= -= \*= /=**
- 증감 연산자 : **++ --**
- 논리 연산자 : **! && ||**
- 비교 연산자 : **== === != !\== > >= < <=** 
	- \==은 값만 비교 =\==은 자료형까지 비교한다
- 변수의 타입을 알려주는 연산자 : **typeof**
- null병합 연산자 : **??**
	- 양쪽의 피연산자 중에서 null이나 undefined이 아닌 값을 선택한다
