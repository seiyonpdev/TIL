- 기본 선언
	n = 700

- 출력
	print(n)
	→ 700
	
	print(type( n))
	→ <class 'int'>

- 동시 선언
	x = y = z = 500
	print(x, y, z)
	→ 500 500 500

- 재선언
	var = 75
	var = "Change Value"
	print(var)
	→ Change Value
	print(type(var))
	→ <class 'str'>
