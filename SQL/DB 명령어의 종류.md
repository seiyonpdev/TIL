1. [[DDL]] (Data Definition Language) : 데이터 정의 언어
	테이블이나 개체를 생성, 수정, 삭제
2. [[DML]] (Data Manipulation Language) : 데이터 조작 언어
3. [[DCL]] (Data Control Language) : 데이터 제어 언어

- desc 테이블명
	테이블 구조 조회
- ed 파일명
	파일 수정
- @ 파일명
	파일에 적힌 sql문 적용
- column 컬럼명 format 
	문자인 경우 : a문자길이 (ex. a10)
	숫자인 경우 : 9로 자릿수 표시 (ex. 9999) 금액인 경우 중간에 쉼표 넣을 수 있음 (ex. 999,999)
