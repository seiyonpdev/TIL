- if, elif, else
- 중첩 가능

- #### 기본 형식
	- if grade == "A":
		- print("Very Good")
	- elif grade == "B":
		- print("Good")
	- else:
		- print("Try Harder")

- #### 연산자 우선순위
	- 산술 > 관계 > 논리

- #### 관계 연산자
	- <
	- <=
	- >
	- >=
	- ==
	- !=

- #### 논리 연산자
	- and, or, not
- #### in, not in
	- 15 in a
	- 12 not in b
	- "name" in c
	- "Seoul" in d.values()
