
#### 프로젝트 생성

- 스프링 부트 스타터 사이트에서 스프링 프로젝트 생성

- 프로젝트 선택
	- Project : Gradle Project
	- Spring Boot : (버전)
		- SNAPSHOT : 만들다 만 버전
		- M1 : 아직 정식 출시가 안 된 버전
	- Language : Java
	- Packing : Jar
	- Java : (버전)
- Project Metadata
	- groupId : 보통은 기업명이나 기업 도메인 등을 적는다
	- artifactId : 빌드되어 나오는 결과물 이름
- Dependencies : 가져다 쓰고 싶은 라이브러리를 선택한다
